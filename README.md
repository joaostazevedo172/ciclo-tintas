## 📋 Sobre o Projeto

Site profissional desenvolvido para apresentar a linha completa de produtos da Ciclo Tintas:
- Tintas à base de solvente
- Termoplásticos para sinalização
- Tachões e tachinhas
- Segregadores e prismas de resina

## ✨ Funcionalidades

### 🎨 Design Moderno
- ✅ Logo customizada em texto (CICLO TINTAS)
- ✅ Logo SVG ilustrada na seção "Sobre"
- ✅ Paleta de cores: Cinza Grafite (#2f2f2f) + Amarelo Trânsito (#f3bc2c)
- ✅ Tipografia Montserrat (Google Fonts)

### 📱 Estrutura Completa
1. **Navbar Fixa** - Menu responsivo com logo e links
2. **Hero Section** - Background de rodovia + CTAs principais
3. **Produtos** - 6 cards com imagens reais dos produtos
4. **Diferenciais** - 4 pilares da empresa
5. **Sobre Nós** - História + Logo SVG + Badge "10+ Anos"
6. **Contato** - 3 botões de ação (E-mail, WhatsApp, Telefone)
7. **Footer** - Links e dados da empresa
8. **WhatsApp Flutuante** - Botão fixo com animação pulse

### 🎯 Conversão Otimizada
- ✅ 3 pontos de contato com WhatsApp
- ✅ Botões de ação diretos (E-mail, WhatsApp, Telefone)
- ✅ Cards de produtos com CTA individual
- ✅ Hover effects em todos os elementos

## 🚀 Como Usar

### Opção 1: Hospedagem Estática

Basta fazer upload do arquivo `index.html` para qualquer servidor web:

- **Netlify**: Arraste o arquivo para netlify.com/drop
- **Vercel**: `vercel --prod`
- **GitHub Pages**: Commit para branch `gh-pages`
- **Servidor próprio**: Upload via FTP

### Opção 2: Visualizar Localmente

```bash
# Clone o repositório
git clone [seu-repositorio]

# Abra o arquivo
open index.html
# ou
start index.html
```
